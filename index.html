06 — Extend

Tạo theme mới

Khi Alex muốn try direction khác (vd: "Neon Cosmic", "Imperial Gold"), copy template dưới đây.

3 bước thêm theme mới

Bước 1. Trong CSS, copy block [data-theme="sacred"] rename thành [data-theme="your-name"]. Đổi 11 shades primary/secondary/tertiary/neutral + font + radius theo direction mới.

Bước 2. Trong HTML header, add nút mới: <button class="theme-btn" data-target="your-name" onclick="switchTheme('your-name')">Your Theme</button>

Bước 3. Trong JavaScript, thêm meta cho theme mới vào themeMeta object (tên + tagline + roles).

Template skeleton:

[data-theme="my-theme"] {
  /* Primary */
  --primary-500: #YOUR_HEX;
  /* ... full 50-900 scale */

  /* Secondary */
  --secondary-500: #YOUR_HEX;
  /* ... */

  /* Fonts */
  --font-display: 'YourFont', serif;
  --font-body:    'Inter', sans-serif;

  /* Theme meta */
  --theme-name: 'My Theme';
  --theme-tagline: 'Your tagline here';
}

Generate shade scale từ 1 hex: Dùng tool như tailwindshades.com hoặc uicolors.app — input 1 brand color, output 11 shades. Paste vào CSS xong.
