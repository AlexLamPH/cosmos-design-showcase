# Cosmos Design Showcase

Live preview của Cosmos AI Lab design system — 3 brand themes với theme switcher interactive.

**Live URL**: https://alexlamph.github.io/cosmos-design-showcase/

## 🎨 3 Themes

| Theme | URL param | Vibe |
|-------|-----------|------|
| **Sacred Laboratory** | `?theme=sacred` | Warm gold + Vietnamese cosmic + Cormorant Garamond italic |
| **Cosmic Glass** | `?theme=cosmic` | Refined indigo + glassmorphism + Space Grotesk |
| **Phantom Lab** | `?theme=phantom` | Pure black + Archivo Black + Playfair Display italic + outlined text |

Default theme = Sacred. Switch via top header buttons hoặc URL param `?theme=`.
